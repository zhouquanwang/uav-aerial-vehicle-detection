# 轨迹数据后处理方法

> 无人机航拍车辆检测系统 — 轨迹去噪、平滑、流向聚类算法参考

---

## 一、轨迹去噪

### 1.1 空间离群点去除

| 方法 | 原理 | 参数 | 适用场景 |
|------|------|------|----------|
| **DBSCAN 逐轨迹** | 对每条轨迹点集做 DBSCAN，标记噪声点并剔除 | `eps`=像素阈值, `min_samples`=2~3 | 跳跃野点 |
| **中值滤波** | 滑动窗口取 x,y 坐标中位数替换 | 窗口 3~5 | 脉冲噪声 |
| **均值滤波** | 窗口均值替换，简单快速 | 窗口 3~7 | 轻微抖动 |
| **Isolation Forest** | 无监督异常检测，学习正常分布，剔除异常点 | `contamination`=0.05 | 通用离群检测 |
| **LOF (Local Outlier Factor)** | 基于局部密度比检测局部异常 | `n_neighbors`=20 | 密集场景 |

### 1.2 时间序列约束去噪

| 方法 | 原理 | 参数 |
|------|------|------|
| **速度/加速度阈值** | 相邻帧间位移 > 阈值或加速度突变 → 标记/剔除 | 速度阈：100px, 加速度阈：50px/f² |
| **Kalman 预测残差** | 恒定速度模型预测位置 vs 观测位置对比 | 残差 > 3σ → 标记野点 |
| **Hampel 滤波器** | 滑动窗口内 MAD 检测，对离群值鲁棒 | 窗口 7，阈值 3σ |

### 1.3 轨迹级过滤

| 方法 | 原理 | 当前状态 |
|------|------|----------|
| **轨迹长度过滤** | `len(traj) < min_len` → 丢弃 | ✅ 已实现（min_traj_len=5） |
| **位移过滤** | `displacement < min_disp` → 丢弃静止/短途 | ✅ 已实现（min_displacement=30） |
| **曲率异常检测** | 轨迹整体曲率突变 → 去除合并错误 | 待实现 |
| **起始位置聚类** | 大量轨迹起始在画面边缘相同位置 → 去除检测框碎片 | 待实现 |

### 推荐组合流水线

```
速度阈值粗筛 → Kalman 残差精筛 → 中值滤波平滑 → 轨迹长度/位移过滤丢弃
```

## 二、轨迹平滑

### 2.1 经典滤波平滑

| 算法 | 原理 | 参数 | 特点 |
|------|------|------|------|
| **Savitzky-Golay** | 滑动窗口内多项式拟合，保形平滑 | 窗口 7~15，阶数 2~3 | ⭐ 推荐，保留峰值特征 |
| **Gaussian 平滑** | 权重高斯衰减 | σ=1.0~2.0 | 平滑但会模糊拐点 |
| **指数移动平均 (EMA)** | 权重递归，近大远小 | α=0.3~0.7 | 实时在线，无延迟 |
| **B-spline** | 样条拟合 | knot 间距 | 光滑度高 |

### 2.2 运动模型平滑

| 算法 | 原理 | 特点 |
|------|------|------|
| **Kalman Filter** | 恒定速度/加速度模型，在线平滑+预测一步 | 实时适用，滞后小 |
| **RTS (Rauch–Tung–Striebel)** | Kalman 的后向平滑形态，利用全部观测 | ⭐ 推荐，离线最优 |
| **粒子滤波** | 非高斯非线性模型 | 剧烈运动适用 |

### 2.3 优化类平滑

| 算法 | 原理 | 特点 |
|------|------|------|
| **Moving Least Squares (MLS)** | 局部加权最小二乘拟合 | 自适应局部 |
| **L1 Trend Filtering** | L1 正则化分段线性拟合 | ⭐ 保留真实拐点 |
| **LOESS / LOWESS** | 局部加权回归 | 鲁棒性强 |

### 2.4 深度学习

| 方法 | 特点 |
|------|------|
| LSTM/GRU 序列去噪 | 学习正常轨迹模式，重建平滑 |
| Transformer / AutoEncoder | 大窗口上下文建模 |

### 推荐流水线

```
S-G 滤波(窗7,阶2) → B-spline 插值重采样 → RTS 平滑(可选)
```

## 三、流向聚类统计

### 3.1 基于角度的方法

| 算法 | 原理 | 当前状态 |
|------|------|----------|
| **直方图峰值检测** | 圆形角度直方图 → 高斯平滑 → `find_peaks()` → 每峰=一流向 | ✅ 已实现 |
| **DBSCAN (余弦距离)** | `1-cos(θ₁-θ₂)` 作为距离度量 | ⚠️ 有桥接效应 |
| **K-Means (圆形)** | 圆形数据 K-Means 聚类 | 需预知 K |
| **GMM (von Mises 混合)** | 圆形概率分布拟合，自动确定成分数 | BIC/AIC 选 K |
| **Mean Shift (角度)** | 带宽自适应，自动发现峰个数 | 需调带宽 |

### 3.2 多维特征聚类

| 算法 | 特征 | 优势 |
|------|------|------|
| **OD分区 + 角度** | 起点/终点空间扇区 × 方向角 | 空间+方向双重约束 |
| **DBSCAN (多维)** | 起点x,y + 终点x,y + 角度 + 位移 | 自然分离几何流向 |
| **HDBSCAN** | 同 DBSCAN 但自动选择 `eps` | ⭐ 推荐，无需调参 |
| **谱聚类** | 轨迹相似度矩阵 → 特征分解 → K-Means | 非凸簇好 |
| **Agglomerative** | 层次聚类，自定义轨迹距离 | 可观察聚类树 |

### 3.3 轨迹形状相似度聚类

| 算法 | 距离度量 | 特性 |
|------|----------|------|
| **DTW** | 动态时间规整 | 容忍速度差异 |
| **Hausdorff** | 最大最小点集距离 | 对形状敏感 |
| **LCSS** | 最长公共子序列 | 容忍噪声/缺失 |
| **Fréchet** | 同时考虑走位和顺序 | 几何直观 |
| **EDR / ERP** | 编辑距离变体 | 对噪声鲁棒 |

### 3.4 深度学习聚类

| 方法 | 优势 |
|------|------|
| t-SNE / UMAP → DBSCAN | 降维可视化 + 传统聚类 |
| VAE + GMM | 变分自编码器潜空间 → GMM 聚类 |
| Trajectory Transformer → Clustering | 自监督预训练轨迹表示 → 聚类 |

### 推荐方案

```
针对交叉口场景:
  ★ 角度直方图峰值检测(已实现) + HDBSCAN 多维聚类 双验证
  进阶: 谱聚类(角度 + 起终点) → 自动确定流向数 → 每簇 = 一个方向
```
